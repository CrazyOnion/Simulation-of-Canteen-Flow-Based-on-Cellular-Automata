# 基于元胞自动机模拟北航学二食堂就餐时间人流拥堵状况
考虑到仿真难度，部分因素理想化。例如：  
1.不考虑一个桌子坐不满的情况。  
2.不考虑人会局部短暂移动以主动躲避拥堵。  
3.人前往各个餐口的概率相等。  
4.使用区域约束（如灰色区域）模拟人流排队方向。  
5.人均打饭时间为15秒，用餐时间为12至17分钟，1秒移动1个像素。  
6.忽略前往饮料台的人流。   
7.人员用餐完毕后直接离开食堂，忽略送回餐具的时间。
